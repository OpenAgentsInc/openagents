# OpenAgents

Your agent command center. (wip)

## Tech stack

- [Effect](https://effect.website/)
- [webview-bun](https://github.com/tr1ckydev/webview-bun)

## Getting Started

1. Install dependencies:
   ```bash
   bun install
   ```

2. Run in development mode:
   ```bash
   bun start
   ```

3. Build for production:
   ```bash
   bun run build
   ```
