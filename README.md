# OpenAgents

Your agent command center. (wip)

## Tech stack

- [Electrobun](https://blackboard.sh/electrobun/docs/)
- [Effect](https://effect.website/)

## Getting Started

1. Install dependencies:
   ```bash
   bun install
   ```

2. Run in development mode:
   ```bash
   bun start
   ```

3. Build for production:
   ```bash
   bun run build
   ```
