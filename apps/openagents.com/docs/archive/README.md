# Internal documentation archive

This directory preserves tracked historical material that is intentionally
outside the public OpenAgents documentation graph.

Archive contents are visible in Git and may be used as internal historical
reference. They must not be imported by `apps/start/content/docs`, rendered at
`/docs`, copied into `apps/start/public/docs`, indexed by documentation search,
or linked by public documentation.

`desktop-future/` contains the former Future / Advanced reading set, removed
from the public docs on 2026-07-15.
