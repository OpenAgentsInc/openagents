# Slack Integration Notes

This folder records OpenAgents product surface planning notes for Slack workspace authorization,
message ingestion, workroom context, source authority, and Slack-facing
Autopilot features.

- `2026-06-06-slack-team-message-ingestion-audit.md` audits how OpenAgents product surface should
  let a user authorize the OpenAgents Slack bot, prioritize ingest for opted-in
  private channels where the user adds the bot, connect those messages to
  team/project workrooms, and expand into Slack-native Autopilot features.
