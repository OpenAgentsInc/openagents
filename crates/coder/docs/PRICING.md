# Coder Pricing

Simple: seats + monthly credits. Credit card by default. Optional Bitcoin payouts for marketplace builders.

---

## Credit-Based Pricing

Credits are the universal currency for Coder usage:

| Resource | Credit Cost |
|----------|-------------|
| Agent prompt (2K tokens) | 1 credit |
| Workflow run (per step) | 1 credit |
| Deploy (preview) | 5 credits |
| Deploy (production) | 10 credits |
| Storage (GB/month) | 1 credit |

### Overage Rates

When credits are exhausted:
- Pro: $0.10 per credit
- Team: $0.08 per credit
- Business: $0.06 per credit
- Enterprise: Custom
