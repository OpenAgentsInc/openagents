# Commander

Desktop UI for OpenAgents swarm compute network.

## Quick Start

```bash
# From root directory
cargo run --bin commander

# Or use the shortcut
cargo commander
```

## Features

- GPUI-powered GPU-accelerated UI
- Native macOS application
- Real-time agent monitoring
- Bitcoin wallet integration
- Swarm compute management

## Development

```bash
cd crates/commander
cargo run
```

## Requirements

- macOS 10.15+
- Rust 1.70+
- Apple Silicon recommended for best performance
