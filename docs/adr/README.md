# ADR Index

This directory contains architecture decision records for retained MVP implementation doctrine.

- `ADR-0001-spacetime-domain-authority-matrix.md`: defines domain-scoped authority boundaries for Spacetime reintroduction.
- `ADR-0002-provider-presence-cardinality-and-ttl-policy.md`: defines canonical `providers_online` cardinality and heartbeat/TTL semantics.
- `ADR-0003-compute-market-ownership-and-authority-split.md`: defines the owner split between desktop UX, provider substrate, Psionic execution, kernel authority, and planned validator/environment surfaces for the compute market program.
