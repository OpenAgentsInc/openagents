## Methodology

APM (Actions Per Minute) measures agent velocity using the formula:

```
APM = (messages + tool_calls) / duration_minutes
```

**APM Tiers:**
| Range | Tier | Description |
|-------|------|-------------|
| 0-5 | Baseline | Human-in-the-loop, lots of thinking time |
| 5-15 | Active | Steady work pace |
| 15-30 | Productive | Efficient autonomous execution |
| 30-50 | High Performance | Fast parallel tool usage |
| 50+ | Elite | Maximum efficiency |

## Instructions for Second Computer (Linux)

Run this script on your Linux machine from the openagents repo directory:

```bash
#!/bin/bash
# APM Data Collection Script for Linux
# Run from: ~/code/openagents

echo "=== APM DATA COLLECTION (Linux) ==="
echo "Date: $(date -Iseconds)"
echo "Hostname: $(hostname)"
echo ""

# Claude Code data
CLAUDE_DIR="$HOME/.claude/projects/-home-christopherdavid-code-openagents"
if [ -d "$CLAUDE_DIR" ]; then
  echo "=== CLAUDE CODE ==="
  cd "$CLAUDE_DIR"

  total_msgs=0
  total_tools=0
  total_sessions=0
  last48h_msgs=0
  last48h_tools=0
  last48h_sessions=0
  cutoff=$(date -d "2 days ago" +%Y-%m-%d)

  for f in *.jsonl; do
    [ -f "$f" ] || continue

    msg_count=$(grep -c '"type":"message"' "$f" 2>/dev/null || echo 0)
    tool_count=$(grep -c '"type":"tool_use"' "$f" 2>/dev/null || echo 0)
    mod_date=$(stat -c "%y" "$f" 2>/dev/null | cut -d' ' -f1)

    if [ "$msg_count" -gt 0 ] || [ "$tool_count" -gt 0 ]; then
      total_msgs=$((total_msgs + msg_count))
      total_tools=$((total_tools + tool_count))
      total_sessions=$((total_sessions + 1))

      if [[ "$mod_date" > "$cutoff" ]]; then
        last48h_msgs=$((last48h_msgs + msg_count))
        last48h_tools=$((last48h_tools + tool_count))
        last48h_sessions=$((last48h_sessions + 1))
      fi
    fi
  done

  echo "Sessions: $total_sessions"
  echo "Messages: $total_msgs"
  echo "Tool calls: $total_tools"
  echo "Total actions: $((total_msgs + total_tools))"
  echo "Last 48h sessions: $last48h_sessions"
  echo "Last 48h actions: $((last48h_msgs + last48h_tools))"
else
  echo "Claude Code directory not found at: $CLAUDE_DIR"
fi

# Autopilot data
echo ""
echo "=== AUTOPILOT ==="
cd ~/code/openagents
if [ -d "docs/logs" ]; then
  total_tools=0
  total_sessions=0
  last48h_tools=0
  last48h_sessions=0
  cutoff=$(date -d "2 days ago" +%Y-%m-%dT%H:%M:%SZ)

  for dir in docs/logs/2025*/; do
    [ -d "$dir" ] || continue
    for f in "$dir"*.rlog; do
      [ -f "$f" ] || continue

      start_ts=$(grep -m1 "^@start" "$f" 2>/dev/null | sed 's/.*ts=\([^ ]*\).*/\1/')
      tools=$(grep -c "^t!:" "$f" 2>/dev/null || echo 0)

      if [ "$tools" -gt 0 ] && [ -n "$start_ts" ]; then
        total_tools=$((total_tools + tools))
        total_sessions=$((total_sessions + 1))

        if [[ "$start_ts" > "$cutoff" ]]; then
          last48h_tools=$((last48h_tools + tools))
          last48h_sessions=$((last48h_sessions + 1))
        fi
      fi
    done
  done

  echo "Sessions: $total_sessions"
  echo "Tool calls: $total_tools"
  echo "Last 48h sessions: $last48h_sessions"
  echo "Last 48h tool calls: $last48h_tools"
else
  echo "docs/logs directory not found"
fi

echo ""
echo "=== COPY THE OUTPUT ABOVE INTO Part 2 OF THE REPORT ==="
```

Save as `collect-apm.sh`, make executable (`chmod +x collect-apm.sh`), and run it.

## Observations

1. **APM is lower than expected** - The directive predicted ~4.5 APM for Claude Code and ~19 APM for Autopilot. Actual measurements show ~5.6 and ~6.6 respectively. This could be due to:
   - Duration estimates including idle time
   - Sessions that had errors/restarts
   - Different workload patterns

2. **Claude Code has 8.5x more total actions** than Autopilot (155K vs 18K) because it's been running longer and has more sessions.

3. **Autopilot 48h APM (6.9) is higher than lifetime (6.6)** suggesting recent sessions have been more efficient.

4. **Week 49 (Dec 8-14) was the peak** with 847 Claude Code sessions - likely a major feature push.
