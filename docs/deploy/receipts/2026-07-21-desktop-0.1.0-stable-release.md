## What shipped

The first stable OpenAgents Desktop release was rebuilt from a clean `0.1.0`
version (the stable channel refuses a prerelease-versioned ReleaseSet), signed,
notarized, published, and promoted to the stable channel through the owned
release coordinator. The public feed serves it, and `/download` resolves it.

### Build + sign

All four required targets were rebuilt at the `0.1.0` commit:

- `darwin-arm64`, `darwin-x64`: built locally (arm64 native, x64 cross-build),
  signed with Developer ID `HQWSG26L43`, notarized (Apple: Accepted), stapled,
  and Gatekeeper-verified (`spctl` reports `source=Notarized Developer ID`).
- `linux-x64`: built on the standing GCE worker `oa-rel-worker-linux-x64`.
- `linux-arm64`: built on `oa-rel-worker-linux-arm64` (started for the build,
  stopped after).
- All six Linux artifacts (AppImage/DEB/RPM per architecture) were signed with
  the ed25519 release key (kid `2dbe811d19f67528`) on a host isolated from the
  build workers, and each passes fail-closed `verify-release` against the pinned
  public key.
