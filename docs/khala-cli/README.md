# Khala CLI (retired)

The `@openagentsinc/khala` client and `clients/khala-cli` source were retired
and removed by owner direction on 2026-07-14. Do not install, publish, or revive
the package. Use Pylon for supported terminal and Codex-capacity workflows:

```sh
npm install -g @openagentsinc/pylon
pylon auth codex
pylon accounts list --json
```

The dated documents in this directory are historical design and release
evidence. Recover deleted source from the pre-removal commit recorded in
[`../sol/2026-07-14-clients-retirement-after-action.md`](../sol/2026-07-14-clients-retirement-after-action.md).
