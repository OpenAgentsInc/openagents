# OpenAgents Desktop MVP assurance coverage matrix

This is the human-readable projection of the checked-in UX-5 congruence oracle. The authority is `mvpDockSurfaces`; every expected-working surface interaction maps to ProductSpec intent, an AssuranceSpec item, an enforced behavior contract, and an executable oracle. Coverage outside that allowlist and missing coverage both fail the normal Desktop sweep.
