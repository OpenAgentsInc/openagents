## Reading rule

- Use the final-disposition header before intermediate body sections.
- Preserve intermediate failures and partial rungs; they explain why the final
  contract exists.
- A closed leaf does not close its parent or a different live/device rung by
  implication.
- When this index and a live issue differ, refresh the live issue and reconcile
  this index; do not silently rewrite the receipt.
