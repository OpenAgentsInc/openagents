# Psionic Intro Dataset

This is a small sample Data Market dataset for the OpenAgents `v0.2` data
market MVP.

It is meant to answer a simple first question:

What is Psionic, what does it own, and where should a new reader start?

The bundle is intentionally small and introductory. It does not try to mirror
the full `psionic` repository. Instead it curates a few short derived guides
from the current canonical Psionic docs and records the exact source material
used to build them.

Included files:

- `01-what-is-psionic.md`
- `02-system-architecture.md`
- `03-inference-training-and-backends.md`
- `04-openagents-boundary.md`
- `provenance.json`
