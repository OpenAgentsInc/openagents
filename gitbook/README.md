# Welcome to OpenAgents

> _"OpenAgents exists to solve one problem. **America has no frontier open-source AI lab.**"_
>
> — Christopher David, [Bitcoinfi Demo Day](assets/clips/cdavid-demoday-highlight-90s.mp4)

OpenAgents is America's leading open-source AI lab — proudly based in Austin, Texas, the Bitcoin capital of the world. We build the economic infrastructure for machine work: the [Autopilot](investors/03-autopilot-wedge.md) desktop app on top, [five interlocking markets](investors/02-five-markets.md) — Compute, Data, Labor, Liquidity, Risk — on an [Economy Kernel](investors/07-economy-kernel.md) underneath, all settled in Bitcoin over Lightning and Nostr.

## The canon

Everything in this book is anchored to four external sources:

1. The public repo — [`OpenAgentsInc/openagents`](https://github.com/OpenAgentsInc/openagents)
2. Christopher David's Demo Day pitch — [bitcoinfi.network/demoday](https://bitcoinfi.network/demoday)
3. The OAPN podcast — [openagents.substack.com](https://openagents.substack.com) and [YouTube](https://www.youtube.com/@OpenAgentsInc)
4. Signed release and proof receipts in [`docs/reports/`](https://github.com/OpenAgentsInc/openagents/tree/main/docs/reports)

If any claim in any chapter drifts from those sources, the sources are authoritative.

## How this book is organized

- **Pathway-first** — every reader picks one of three paths on this page.
- **Repo-grounded** — every number, version tag, and quoted phrase is a link back to a specific file, commit, or receipt.
- **Soft ask** — we don't publish seed-round figures or terms in this book. Conversations happen directly.
