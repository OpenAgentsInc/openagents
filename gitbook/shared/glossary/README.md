# Glossary

Short, definitional entries for the OpenAgents terminology that appears across all three pathways. Each entry is ~200 words and links back to the chapters where the term is used in context.

## Markets

- [**Compute Market**](compute-market.md) — The lane where paid machine work (inference, embedding, fine-tuning, training) is priced, matched, validated, and settled in Bitcoin.
- [**Risk Market**](risk-market.md) — The underwriting layer that prices uncertainty across Compute, Data, Labor, and Liquidity.

## Kernel & authority

- [**TreasuryRouter**](treasury-router.md) — The backend authority lane that cuts, signs, and settles accepted-work payouts.

## More terms

The full taxonomy is distributed across the [investor chapters](../../investors/README.md) — look for the first use of each term, which is where it is defined. Terms worth expanding into standalone glossary entries in the future include **Autopilot**, **Pylon**, **Nexus**, **Psionic**, **Spacetime**, **CS336 A1**, **NIP-90**, and **Economy Kernel**.
