## What's available today

The provider binary that Autopilot wraps — **Pylon** — is published on npm and can be installed with one command:

```bash
npx @openagentsinc/pylon@0.1.13 --help
```

Release receipt: [`pylon-v0.1.13`](https://github.com/OpenAgentsInc/openagents/releases/tag/pylon-v0.1.13). SHA-256 for the `darwin-arm64` binary: `de995efc90675d90108785a2790e0c2bc4099cd0ef6eaff2d8ae58fccc234a66`.
