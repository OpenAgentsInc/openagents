# @openagentsinc/theme

Shared theme tokens and web CSS for OpenAgents apps.

- Colors: `import { Colors } from '@openagentsinc/theme'`
- Typography tokens: `import { Typography } from '@openagentsinc/theme'`
- Web CSS variables: `import '@openagentsinc/theme/web/theme.css'`

This package does not include platform-specific font loaders. Expo should continue to load fonts via `expo-font`.
