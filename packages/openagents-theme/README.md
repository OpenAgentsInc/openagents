# @openagents/theme

Shared theme tokens and web CSS for OpenAgents apps.

- Colors: `import { Colors } from '@openagents/theme/colors'`
- Typography tokens: `import { Typography } from '@openagents/theme/typography'`
- Web CSS variables: `import '@openagents/theme/web/theme.css'`

This package does not include platform-specific font loaders. Expo should continue to load fonts via `expo-font`.
